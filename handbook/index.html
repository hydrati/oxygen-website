# 概述

# 扯淡

大约在一年前吧，有个朋友(@「杉」) 跟我说有个PE(Edgeless (opens new window))很不戳，让我试试。我一看，噢，一个 Windows10 PE，600多MB，没甚么稀奇的，我放VM里一看，不得了：来，脚本！ 来，插件！来，主题！把我这个才10多岁的小同志吓得不轻啊。这是我见也没有见过的，我连忙去看了文档，噢，可以合作啊(Edgeless - 合作洽谈 (opens new window))。我连忙去找了作者(@Cno)，他嗦这个可以有，但是先要实现了再说。我还愁着怎么移植，噢，我一看，原来有个Edgeless ME (opens new window)啊，我下载下来，加载PECMD脚本，一气呵成，我又放进VM里启动一看，欸？？？怎么没有主题包？？？？后来我一看，主题模块闭源(Edgeless Repo - 许可证声明 (opens new window))，但可以复刻啊！我打开 Edgeless，经过 1h23min 的 Coding，诶，终于码出来了，我发给Cno，他说不得了，这个人有操作的啊...

Oxygen Project 是 Oxygen Team在 2020/3/20 就开始酝酿的 PE 项目，从 2020/09 开始，这个项目就暂停开发了（学业原因）。直到2021/1，项目重新开始。

# 为什么要做这玩意?

纯属好玩（大雾）

嘶...我感觉目前市面上的PE感jio可自定义性太低了，而且可以自定义的也没有规范的接口（如 Edgeless (opens new window)）。

既然这么想要自定义，那就把自定义贯彻到底吧~~~~

然后就动手测试了下可行性，将Windows PE的Explorer外置，居然可以用（狂喜）。 于是，就有了这个项目。
